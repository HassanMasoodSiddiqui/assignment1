The deployment of this website was done using netlify. Netlify is a simple to use web hosting service which is free to use for static webpages i.e. webpages that only utilize HTML, CSS and javascript without using any database structure. Netlify makes the task of hosting the website really simple as all the steps you have to take are as easy as:

  1. Creating an account on netlify
  2. After navigating to the deploy screen you just drag and drop your website folder there making sure the ‘index.html’ file is present inside the root directory
  3. And that’s it netlify generates a random URL for your site and gives it to you which you can then use to access your site on the internet
If you already own a custom domain you can hook it up to your site too other than that netlify also provides security to your site by adding https using SSL certificate
